Розклад на місяць (-15 і +15 днів від сьогодні)
(для розкладу на тиждень треба змінити 30 на 7)

SELECT schedule.DATE, schedule.LESSON_POSITION, lesson.NAME, teacher.FIRSTNAME, teacher.LASTNAME,
classroom.NAME, schedule.HOMEWORK
FROM schedule 
JOIN teacher ON teacher.ID = schedule.TEACHER_ID
JOIN classroom ON schedule.CLASSROOM_ID = classroom.ID
JOIN lesson ON schedule.LESSON_ID = lesson.ID
WHERE schedule.FORM_ID = (my form ID here)
AND TIMESTAMPDIFF(DAY,schedule.DATE,DATE_FORMAT(now(), '%Y/%m/%d')) < 30
ORDER BY schedule.date, schedule.LESSON_POSITION 

Я хочу бачити свої оцінки з минулих уроків, а також відмітки відвідуваності.

SELECT SCHEDULE.DATE, LESSON.NAME, ATTENDANCES.ATTEND, ATTENDANCES.GRADE
FROM SCHEDULE
JOIN LESSON ON SCHEDULE.LESSON_ID = LESSON.ID
JOIN ATTENDANCES ON SCHEDULE.ID = ATTENDANCES.SCHEDULE_ID
WHERE ATTENDANCES.PUPIL_ID IN(
SELECT ID FROM PUPIL 
WHERE FIRSTNAME LIKE 'Вовочка' AND LASTNAME LIKE 'Іванов') AND DATE < now()
ORDER BY SCHEDULE.DATE, SCHEDULE.LESSON_POSITION;
